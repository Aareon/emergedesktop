About

emergeMediaCtrl provides a 'remote control' applet interface to various media players including WinAmp, foobar and Media Player Classic.

Configuration

emergeMediaCtrl settings are stored in HKEY_CURRENT_USER\Software\Emerge Desktop\emergeMediaCtrl and consist of the following subkeys:

  • Settings - all emergeMediaCtrl related settings. Currently the options are:
    • X - The position of the window on the x-axis (Default 0)
    • Y - The position of the window on the y-axis (Default 0)
    • ActiveWindowAlpha - The degree of alpha transaprency of the window (Default 100) when the mouse if over the window
    • InactiveWindowAlpha - The degree of alpha transaprency of the window (Default 80) when the mouse if off the window
    • ZPosition - The position of the window on the z-axis (window order - Default "normal")
    • Prev - The icon to use for the 'Previous' button (Default "prev.ico")
    • Pause - The icon to use for the 'Pause' button (Default "pause.ico")
    • Stop - The icon to use for the 'Stop' button (Default "stop.ico")
    • Next - The icon to use for the 'Next' button (Default "next.ico")
    • Launch - The icon to use for the 'Launch' button (Default "launch.ico")
    • Play - The icon to use for the 'Play' button (Default "play.ico")
    • Open - The icon to use for the 'Open' button (Default "open.ico")
    • Player - The player to control (winamp, foobar or mpc - Default "foobar")

Example configuration:

[HKEY_CURRENT_USER\Software\Emerge Desktop\emergeMediaCtrl]

[HKEY_CURRENT_USER\Software\Emerge Desktop\emergeMediaCtrl\Settings]
"X"=dword:00000338
"Y"=dword:000002e2
"ActiveWindowAlpha"=dword:00000050
"InactiveWindowAlpha"=dword:0000001e
"ZPosition"="Top"

Usage

Left clicking on the buttons performs the actions assigned to them (i.e. the "Play" button plays, etc.)

Holding down 'CTRL' while left clicking on the window will allow it to be dragged around the desktop. When the window nears the screen edge, it will snap to it, also, it will not allow itself to be dragged offscreen. By default, the new position is stored in the X and Y settings in the registry so that they will be remebered on next start. To bypass saving the new position, holding down 'Shift' while moving the window.
