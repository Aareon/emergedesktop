Command Line Switches

The following is a list of Command line switches supported by Emerge Desktop:

  • /nostartup - Stops Emerge from running at startup programs included in the Startup Group, in HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run and in HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Run
  • /shellchanger - Display the Shell Changer dialog. This allows to change to/from Emerge Desktop from/to other shells (including explorer) or add new shells to the list. You also have access to this dialog from the Settings menu or by typing the CoreShellChanger internal command in emergeCommand

E.g. You can type in a Run box or command line "C:\Program Files\Emerge Desktop\emergeCore.exe" /shellchanger (don't forget to add quotes if the path to emergeCore has spaces)
