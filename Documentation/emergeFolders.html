About

emergeFolders displays the contents of specific folders in an applet window.

Configuration

emergeFolders settings are stored in HKEY_CURRENT_USER\Software\Emerge Desktop\emergeFolders and consist of the following subkeys:

  • Settings - all emergeFolders related settings. Currently the options are:
    • X - The position of the window on the x-axis (Default 0)
    • Y - The position of the window on the y-axis (Default 0)
    • ActiveWindowAlpha - The degree of alpha transaprency of the window (Default 100) when the mouse if over the window
    • InactiveWindowAlpha - The degree of alpha transaprency of the window (Default 80) when the mouse if off the window
    • ZPosition - The position of the window on the z-axis (window order - Default "normal")
    • Path1 - The path to the folder1 you want to display
    • Icons1 - The icon to be used for this folder1
    • Tip1 - The tooltip to be associated with this folder1
    • Path2 - The path to the folder2 you want to display
    • Icons2 - The icon to be used for this folder2
    • Tip2 - The tooltip to be associated with this folder2
    • Path3 - The path to the folder3 you want to display
    • Icons3 - The icon to be used for this folder3
    • Tip3 - The tooltip to be associated with this folder3
    • Path4 - The path to the folder4 you want to display
    • Icons4 - The icon to be used for this folder4
    • Tip4 - The tooltip to be associated with this folder4
    • Path5 - The path to the folder5 you want to display
    • Icons5 - The icon to be used for this folder5
    • Tip5 - The tooltip to be associated with this folder5
    • Path6 - The path to the folder6 you want to display
    • Icons6 - The icon to be used for this folder6
    • Tip6 - The tooltip to be associated with this folder6
    • Path7 - The path to the folder7 you want to display
    • Icons7 - The icon to be used for this folder7
    • Tip7 - The tooltip to be associated with this folder7

Usage

Left clicking on the specific icon will display the folder contents withing the applet workspace window. Clicking the arrow in the top right toggles the applet workspace window.

Holding down 'CTRL' while left clicking on the window will allow it to be dragged around the desktop. When the window nears the screen edge, it will snap to it, also, it will not allow itself to be dragged offscreen. By default, the new position is stored in the X and Y settings in the registry so that they will be remebered on next start. To bypass saving the new position, holding down 'Shift' while moving the window.
