About

emergeCore, as it name indicates, is the core component of Emerge Desktop. It is the messaging interface between all the other Emerge Desktop applets.

It also launches all startup programs as well as starting the Microsoft DDE Server.

Configuration

There are two major components of Emerge Desktop which are dependent on the Core: the applets loaded at Startup which are configured by the Launch Applets Configuration and the Shell Changer which allows to set Emerge Desktop (or any other alternate shell) as the default shell for the Current User or to switch to Windows Explorer.

To display the Launch Applets Configuration dialog the user should click with the right mouse button on the desktop window, and select Settings, Edit Launch Applets. Alternatively the user can type CoreLaunchEditor on the emergeCommand command line.

To Add items to the Launch list, the user must Browse to the file and then click on Save to save the item or Clear to start again.
To delete a list item, the user should select it from the list and press the Delete button.
The Up and Down buttons allow the user to move an applet up or down in the launch list, to sort the launching order.
The Launch list can also be used to Stop (i.e. unload or close) or Start (i.e. load or open) an applet by selecting it from the list and pressing the apropriate button.

To display the Desktop Shell dialog the user should click with the right mouse button on the desktop window, and select Settings, Change Desktop Shell. Alternatively the user can use the shortcut named "Emerge Desktop Shell Changer" included in the Emerge Desktop group added to the Startup menu during installation. This shortcut can be accessed both when Windows Explorer or Emerge Desktop is the shell. Finally, the user can type CoreShellChanger on the emergeCommand command line.

Emerge Desktop and Windows Explorer are included by default in the Shell list (in fact they can not be removed from it). However the user can Add

other shells to the list, by Browsing to the file, defining a Name, and then clicking on Save to save the item or Clear to start again.
The added shells can be modified or deleted by selecting it from the drop list and pressing the Modify or Delete button.

To switch from one shell to another, the user should select the shell to be used on the next session from the drop list and click on the Change button. After rebooting or logoff, Windows will load with the selected shell.

emergeCore settings are stored in HKEY_CURRENT_USER\Software\Emerge Desktop\emergeCore and consist of the following subkeys:

  • Settings - all emergeCore related settings. Currently the options are:
    • InstallDir - Path where Emerge Desktop was installed
  • Launch - stores a list of strings containing the applications to be launched when emergeCore executes. The values are of type String the name is up to the user (the order is dependent on the name used) and the data is the either the full path to the application to be launched or the path relative to the directory in which emergeCore exists.
  • Shells - stores a list of strings containing alternate shells to be launched. Explorer and Emerge Desktop are not listed although they show in the Shell Changer

Usage

emergeCore also supports command line switches (see Emerge Desktop Command Line Switches).
