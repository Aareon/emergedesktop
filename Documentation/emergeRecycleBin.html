About

emergeRecycleBin provides an interface to the Recycle Bin.

Configuration

emergeRecycleBin settings are stored in HKEY_CURRENT_USER\Software\Emerge Desktop\emergeRecycleBin and consist of the following subkeys:

  • Settings - all emergeRecycleBin related settings. Currently the options are:
    • X - The position of the window on the x-axis (Default 0)
    • Y - The position of the window on the y-axis (Default 0)
    • ActiveWindowAlpha - The degree of alpha transaprency of the window (Default 100) when the mouse if over the window
    • InactiveWindowAlpha - The degree of alpha transaprency of the window (Default 80) when the mouse if off the window
    • ZPosition - The position of the window on the z-axis (window order - Default "normal")

Example configuration:

[HKEY_CURRENT_USER\Software\Emerge Desktop\emergeRecycleBin]

[HKEY_CURRENT_USER\Software\Emerge Desktop\emergeRecycleBin\Settings]
"X"=dword:00000338
"Y"=dword:000002e2
"ActiveWindowAlpha"=dword:00000050
"InactiveWindowAlpha"=dword:0000001e
"ZPosition"="Top"

Usage

Double left clicking on the applet will display the 'Recycle Bin' window. Left clicking on the Recycle Bin icons will empty the bin if there are items in it.

Holding down 'CTRL' while left clicking on the window will allow it to be dragged around the desktop. When the window nears the screen edge, it will snap to it, also, it will not allow itself to be dragged offscreen. By default, the new position is stored in the X and Y settings in the registry so that they will be remebered on next start. To bypass saving the new position, holding down 'Shift' while moving the window.
