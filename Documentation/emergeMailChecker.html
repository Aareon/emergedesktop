About

emergeMailChecker provides a POP3 mail checking applet for Emerge Desktop

Configuration

emergeMailChecker settings are stored in HKEY_CURRENT_USER\Software\Emerge Desktop\emergeMailChecker and consist of the following subkeys:

  • Settings - all emergeMailChecker related settings. Currently the options are:
    • X - The position of the window on the x-axis (Default 0)
    • Y - The position of the window on the y-axis (Default 0)
    • ActiveWindowAlpha - The degree of alpha transaprency of the window (Default 100) when the mouse if over the window
    • InactiveWindowAlpha - The degree of alpha transaprency of the window (Default 80) when the mouse if off the window
    • ZPosition - The position of the window on the z-axis (window order - Default "normal")
    • MailClient - The preferred mail client
    • ClientIcon - The icon to use for the client (Note: is none is specifed the default icon of the mail client is used)
    • User - The POP3 user ID
    • Password - The POP3 password
    • Server - The POP3 server hostname or IP address
    • PollTime - The pause between checking the server for mail in minutes (Default 10)

Example configuration:

[HKEY_CURRENT_USER\Software\Emerge Desktop\emergeMailChecker]

[HKEY_CURRENT_USER\Software\Emerge Desktop\emergeMailChecker\Settings]
"X"=dword:00000338
"Y"=dword:000002e2
"ActiveWindowAlpha"=dword:00000050
"InactiveWindowAlpha"=dword:0000001e
"ZPosition"="Top"
"MailClient"="C:\\Program Files\\Mozilla Thunderbird\\thunderbird.exe"
"ClientIcon"=""
"User"=""
"Password"=""
"Server"="pop3"
"PollTime"=dword:0000000a

Usage

Double left clicking on the applet brings up the configuration dialog. Left clicking on the 'Mail Client' will launch the predefined mail client.

Holding down 'CTRL' while left clicking on the window will allow it to be dragged around the desktop. When the window nears the screen edge, it will snap to it, also, it will not allow itself to be dragged offscreen. By default, the new position is stored in the X and Y settings in the registry so that they will be remebered on next start. To bypass saving the new position, holding down 'Shift' while moving the window.
