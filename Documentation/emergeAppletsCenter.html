About

iAppletCenter allows for the starting and stopping of the individual Emerge Desktop applets.

Configuration

iAppletCenter settings are stored in HKEY_CURRENT_USER\Software\Emerge Desktop\iAppletCenter and consist of the following subkeys:

  • Settings - all iAppletCenter related settings. Currently the options are:
    • X - The position of the window on the x-axis (Default 0)
    • Y - The position of the window on the y-axis (Default 0)
    • ActiveWindowAlpha - The degree of alpha transaprency of the window (Default 100) when the mouse if over the window
    • InactiveWindowAlpha - The degree of alpha transaprency of the window (Default 80) when the mouse if off the window
    • ZPosition - The position of the window on the z-axis (window order - Default "normal")

Example configuration:

[HKEY_CURRENT_USER\Software\Emerge Desktop\iAppletCenter]

[HKEY_CURRENT_USER\Software\Emerge Desktop\iAppletCenter\Settings]
"X"=dword:00000338
"Y"=dword:000002e2
"ActiveWindowAlpha"=dword:00000050
"InactiveWindowAlpha"=dword:0000001e
"ZPosition"="Top"

Usage

Left clicking on the drop down arrow will display a list of the Emerge Desktop applets. Left clicking on one of the applet names will stop the applet if it's running, or start it if it is not.

Ctrl-Left clicking the window will allow for the moving of emergeAppletsCenter around the desktop window. emergeAppletsCenter will remember it's placement when moved. Holding down Shift while moving the window will be a temporary move, meaning emergeAppletsCenter will not remember it's new placement.
