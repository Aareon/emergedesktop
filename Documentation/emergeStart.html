About

emergeStart provides an interface to the Emerge Desktop menus and functions.

Configuration

emergeStart settings are stored in HKEY_CURRENT_USER\Software\Emerge Desktop\emergeStart and consist of the following subkeys:

  • Settings - all emergeStart related settings. Currently the options are:
    • X - The position of the window on the x-axis (Default 0)
    • Y - The position of the window on the y-axis (Default 0)
    • ActiveWindowAlpha - The degree of alpha transaprency of the window (Default 100) when the mouse if over the window
    • InactiveWindowAlpha - The degree of alpha transaprency of the window (Default 80) when the mouse if off the window
    • ZPosition - The position of the window on the z-axis (window order - Default "normal")
    • Layout - The position to use for the menus (Default "Top")
    • Icon1 - The icon to use in the first button (Default "emergeConfig.ico")
    • Command1 - The command to execute for the first button (Default 1 - for emergeConfig)
    • Icon2 - The icon to use in the second button (Default "quit.ico")
    • Command2 - The command to execute for the second button (Default 2 - for Quit)
    • Icon3 - The icon to use in the third button (Default "menu.ico")
    • Command3 - The command to execute for the third button (Default 3 - for MidMenu)
    • Icon4 - The icon to use in the fourth button (Default "lock.ico")
    • Command4 - The command to execute for the fourth button (Default 4 - to lock the computer)

Example configuration:

[HKEY_CURRENT_USER\Software\Emerge Desktop\emergeStart]

[HKEY_CURRENT_USER\Software\Emerge Desktop\emergeStart\Settings]
"X"=dword:00000338
"Y"=dword:000002e2
"ActiveWindowAlpha"=dword:00000050
"InactiveWindowAlpha"=dword:0000001e
"ZPosition"="Top"
"layout"="none"
"icon1"=""
"icon2"=""
"icon3"=""
"icon4"=""
"command1"=dword:00000001
"command2"=dword:00000002
"command3"=dword:00000003
"command4"=dword:00000004

Usage

Left clicking on each of the button performs the actions described in the tooltips.

Holding down 'CTRL' while left clicking on the window will allow it to be dragged around the desktop. When the window nears the screen edge, it will snap to it, also, it will not allow itself to be dragged offscreen. By default, the new position is stored in the X and Y settings in the registry so that they will be remebered on next start. To bypass saving the new position, holding down 'Shift' while moving the window.
