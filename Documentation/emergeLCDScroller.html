About

emergeLCDScroller displays an RSS feed in an LCD type scolling window.

Configuration

emergeLCDScroller settings are stored in HKEY_CURRENT_USER\Software\Emerge Desktop\emergeLCDScroller and consist of the following subkeys:

  • Settings - all emergeLCDScroller related settings. Currently the options are:
    • X - The position of the window on the x-axis (Default 0)
    • Y - The position of the window on the y-axis (Default 0)
    • ActiveWindowAlpha - The degree of alpha transaprency of the window (Default 100) when the mouse if over the window
    • InactiveWindowAlpha - The degree of alpha transaprency of the window (Default 80) when the mouse if off the window
    • ZPosition - The position of the window on the z-axis (window order - Default "normal")
    • ScrollTime - The pause between scrolling (Default 600)
    • LoadURL - Load information from URL (Default 0)
    • URL - URL to be displayed (Default "http://ishell.sourceforge.net/e107_files/backend/news.txt")
    • TextFile - Path to file to use as input (Default "(emergeLCDScroller Path)\rssfeeds.txt")
    • ScreenRes - The screen resolution width (Default 1024)
    • UseProxy - Connect to the internet through a proxy (Default 0)
    • ProxyServer - The hostname or IP address of the proxy server
    • ProxyPort - The port to access on the proxy server (Default 0)

Example configuration:

[HKEY_CURRENT_USER\Software\Emerge Desktop\emergeLCDScroller]

[HKEY_CURRENT_USER\Software\Emerge Desktop\emergeLCDScroller\Settings]
"X"=dword:00000338
"Y"=dword:000002e2
"ActiveWindowAlpha"=dword:00000050
"InactiveWindowAlpha"=dword:0000001e
"ZPosition"="Top"
"scrolltime"=dword:00000258
"screenres"=dword:00000400
"useproxy"=dword:00000001
"proxyserver"="qe70proxy01"
"proxyport"=dword:00000050
"loadurl"=dword:00000001
"url"="http://blizzle.com/?currentTheme=rss"
"textfile"="C:\\Emerge Desktop\\bin\\rssfeeds.txt"
"lastfeedname"="blizzle!"

Usage

Double left clicking on the applet displays the configuration dialog.

Holding down 'CTRL' while left clicking on the window will allow it to be dragged around the desktop. When the window nears the screen edge, it will snap to it, also, it will not allow itself to be dragged offscreen. By default, the new position is stored in the X and Y settings in the registry so that they will be remebered on next start. To bypass saving the new position, holding down 'Shift' while moving the window.
