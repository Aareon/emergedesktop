In order for Emerge Desktop to operate properly, the following directory structure must be used:

..\Emerge Desktop
|
|------ documentation
| |
| |------ Emerge Desktop.chm
|
|------ files
| |
| |------ cmd.txt (alias list for emergeCommand)
|
|------ scripts
| |
| |------ (scripts files in Perl, Python and Ruby that can be used to control Emerge Desktop)
|
|
|------ schemes
| |
| |------ (.eds scheme files for customizing applets)
|
|------ emergeAppletEngine.dll
|------ emergeBaseClasses.dll
|------ emergeCore.exe
|------ emergeDesktop.exe
|------ emergeGraphics.dll
|------ emergeLib.dll
|------ emergeSchemeEngine.dll
|------ emergeTasks.exe
|------ emergeTray.exe
|------ all other Emerge Desktop applets
