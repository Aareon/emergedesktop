Internal Commands

Internal Commands supported by Emerge Desktop:

  • RightDeskMenu - Display the desktop right click menu at the current cursor position
  • MidDeskMenu - Display the desktop middle click menu at the current cursor position
  • Quit - Display the Emerge Desktop Quit dialog
  • Run - Display the Windows Run dialog
  • Shutdown - Display the Windows Shutdown dialog
  • Lock - Lock the workstation or switches to another user profile
  • Hide - Hide all the currently loaded Emerge Desktop applets
  • Show - Show all the currently loaded Emerge Desktop applets
  • VWM_1 - Move to the virtual desktop # 1
  • VWM_2 - Move to the virtual desktop # 2
  • VWM_3 - Move to the virtual desktop # 3 (if available), order is left to right in rows, then down to next row
  • VWM_4 - Move to the virtual desktop # 4 (if available), order is left to right in rows, then down to next row
  • VWM_5 - Move to the virtual desktop # 5 (if available), order is left to right in rows, then down to next row
  • VWM_6 - Move to the virtual desktop # 6 (if available), order is left to right in rows, then down to next row
  • VWM_7 - Move to the virtual desktop # 7 (if available), order is left to right in rows, then down to next row
  • VWM_8 - Move to the virtual desktop # 8 (if available), order is left to right in rows, then down to next row
  • VWM_9 - Move to the virtual desktop # 9 (if available), order is left to right in rows, then down to next row
  • VWMUp - Move to the virtual desktop above the current desktop (if available)
  • VWMDown - Move to the virtual desktop below the current desktop (if available)
  • VWMLeft - Move to the virtual desktop to the left the current desktop (if available)
  • VWMRight - Move to the virtual desktop to the right the current desktop (if available)
  • VWMGather - Gather all the windows to the current desktop
  • EmptyBin - Empty the Recycle Bin
  • Logoff - Log off the current user
  • Reboot - Reboot the system
  • Halt - Halt the system
  • Suspend - Suspend the system
  • Hibernate - Hibernate the system
  • Disconnect - Allow the user to 'disconnect' from a terminal server session without logging off
  • ShowDesktop - Toggle all the application windows minimization/restoration
  • VolumeUp - Increase the sound volume
  • VolumeDown - Decrease the sound volume
  • VolumeMute - Toggle the volume mute
  • DesktopSettings - Display the Desktop Settings
  • CoreLaunchEditor - Display the Launch Editor
  • CoreShellChanger - Display the Shell Changer
  • CoreAbout - Display the About dialog
  • Homepage - Open the default browser on the project's homepage http://emergedesktop.org (if the user is connected to the Internet)
  • Tutorial - Open the default browser on the project's tutorial http://sites.google.com/site/emergedesktop/Home/getting-started/the-basics (if the user is connected to the Internet)
Environmental Variables

Environmental Variables supported by Emerge Desktop:

  • %AppletDir% - Emerge Desktop binaries folder
  • %EmergeDir% - Current User's Emerge Desktop settings folder
  • %ThemeDir% - Current theme folder
  • %Desktop% - Current User's Desktop folder
  • %CommonDesktop% - All Users' Desktop folder
  • %Documents% - Current User's "My Documents" folder
  • %CommonDocuments% - All Users' "My Documents" folder
  • %StartMenu% - Current User's "Start Menu" folder
  • %CommonStartMenu% - All Users' "Start Menu" folder
