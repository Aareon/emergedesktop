Internal Commands

Internal Commands supported by Emerge Desktop:

  • About - Displays the About window.
  • Activate - Opens the emergeHotkeys configuration dialog box with the "emergeHotkeys" parameter, or places a cursor in the emergeCommand window with the "emergeCommand" parameter (For additional instances, see ShowApplet).
  • AliasEditor - Displays the Alias Editor configuration dialog box.
  • CoreSettings -  Displays the Advanced tab of the emergeCore Configuration dialog box.
  • Disconnect - Allows the user to 'disconnect' from a terminal server session without logging off. Disconnects without the dialog box if followed by " /silent".
  • EmptyBin - Empties the Recycle Bin. Empties without the dialog box if followed by " /silent".<
  • Halt - Halts the system. Halts without the dialog box if followed by " /silent".
  • Help - Display the Offline Help.
  • Hibernate - Hibernates the system. Hibernates without the dialog box if followed by " /silent".
  • Hide - Hides all the currently loaded Emerge Desktop applets.
  • Homepage - Opens the default browser on the project's homepage http://emergedesktop.org (if the user is connected to the Internet).
  • LaunchEditor - Displays the Launch Editor.
  • Lock - Locks the workstation or switches to another user profile.
  • Logoff - Logs off the current user. Logs off without the dialog box if followed by " /silent".
  • MidDeskMenu - Displays the desktop middle click menu at the current cursor position.
  • Quit - Displays the Emerge Desktop Quit dialog box. Quits without the dialog box if followed by " /silent".
  • Reboot - Reboots the system. Reboots without the dialog box if followed by " /silent".
  • RightDeskMenu - Displays the desktop right click menu at the current cursor position.
  • Run - Displays the Windows Run dialog.
  • ShellChanger - Displays the Shell Changer.
  • Show - Shows all the currently loaded Emerge Desktop applets.
  • ShowApplet - Toggles the visibility of one Emerge Desktop applet. Must be followed by the name of the applet as parameter. When there is more than one instance of an applet, the name of the additional applet can be found in the title head of the About window of the applet's configuration menu.
  • Shutdown - Displays the Windows Shutdown dialog box.
  • ShowDesktop - Toggles all the application windows minimization/restoration.
  • Suspend - Suspends the system. Suspends without the dialog box if followed by " /silent".
  • Tutorial - Opens the default browser on the project's tutorial http://sites.google.com/site/emergedesktop/Home/getting-started/the-basics (if the user is connected to the Internet).
  • VolumeDown - Decreases the sound volume.
  • VolumeMute - Toggles the volume mute.
  • VolumeUp - Increases the sound volume.
  • VWM_1 - Moves to the virtual desktop # 1.
  • VWM_2 - Moves to the virtual desktop # 2.
  • VWM_3 - Moves to the virtual desktop # 3 (if available), order is left to right in rows, then down to next row.
  • VWM_4 - Moves to the virtual desktop # 4 (if available), order is left to right in rows, then down to next row.
  • VWM_5 - Moves to the virtual desktop # 5 (if available), order is left to right in rows, then down to next row.
  • VWM_6 - Moves to the virtual desktop # 6 (if available), order is left to right in rows, then down to next row.
  • VWM_7 - Moves to the virtual desktop # 7 (if available), order is left to right in rows, then down to next row.
  • VWM_8 - Move to the virtual desktop # 8 (if available), order is left to right in rows, then down to next row.
  • VWM_9 - Moves to the virtual desktop # 9 (if available), order is left to right in rows, then down to next row.
  • VWMDown - Moves to the virtual desktop below the current desktop (if available).
  • VWMGather - Gathers all the windows to the current desktop.
  • VWMLeft - Moves to the virtual desktop to the left the current desktop (if available).
  • VWMNext - Moves to the virtual desktop's next window.
  • VWMPrev - Moves to the virtual desktop's previous window.
  • VWMRight - Moves to the virtual desktop to the right the current desktop (if available).
  • VWMUp - Moves to the virtual desktop above the current desktop (if available).
  • WorkspaceSettings - Displays the Workspace configuration dialog box.

Environmental Variables

Environmental Variables supported by Emerge Desktop:

  • %AppletDir% - Emerge Desktop binaries folder.
  • %EmergeDir% - Current User's Emerge Desktop settings folder.
  • %ThemeDir% - Current theme folder.
  • %Desktop% - Current User's Desktop folder.
  • %CommonDesktop% - All Users' Desktop folder.
  • %Documents% - Current User's "My Documents" folder.
  • %CommonDocuments% - All Users' "My Documents" folder.
  • %StartMenu% - Current User's "Start Menu" folder.
  • %CommonStartMenu% - All Users' "Start Menu" folder.
